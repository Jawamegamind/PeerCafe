How to use this index

- Open any `*-test-doc.md` file to read the test purpose, mock setup, high-level categories, and run instructions.
- Run an individual test file from the `frontend` folder using `npm test -- <path-to-test>`.

Maintenance notes

- When adding new page tests to `frontend/__tests__/pages`, add a corresponding `*-test-doc.md` entry here.
